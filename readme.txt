Create mega menu use the content of a custom post type.

== Description ==

Simple Mega Menu provides a custom post type for creating sub menu markup and some options to choose the markup for the menu item.

== Installation ==

1. Download and extract the zip archive
2. Upload `simple-mega-menu` folder to `/wp-content/plugins/`
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Create the `Mega Menu` post.
5. Edit any menu, turn on `Enable mega menu` for top-level menu item and choose the post you have just created.


== Frequently Asked Questions ==

1. Why does mega menu look weird?
This plugin only adds the mega menu markup and a bit CSS. You need to add your own CSS to make it more beautiful.

2. Can this plugin show category posts with thumbnail on menu?
Yes, it uses the content of post type Mega menu. You can use HTML markup, shortcodes, or page builder to build them.
